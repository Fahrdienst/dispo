# Kurzanleitung für Fahrerinnen und Fahrer

Willkommen beim Fahrdienst. Diese Anleitung zeigt Ihnen die vier wichtigsten Dinge. Schritt für Schritt. Sie brauchen dafür nur Ihr Handy.

Weitere Hilfe finden Sie jederzeit in der App unter **Hilfe**: [/hilfe](/hilfe)

---

## 1. Einladung annehmen und Passwort setzen

Sie bekommen eine E-Mail vom Fahrdienst.

1. Öffnen Sie die E-Mail.
2. Tippen Sie auf den blauen Knopf **Passwort setzen**.
3. Die App öffnet sich.
4. Denken Sie sich ein Passwort aus. Mindestens 12 Zeichen.
5. Geben Sie das Passwort zweimal ein.
6. Tippen Sie auf **Speichern**.

Fertig. Sie sind jetzt angemeldet.

> **Tipp:** Schreiben Sie sich Ihr Passwort auf. Bewahren Sie es sicher auf.

## 2. Verfügbarkeit eintragen

Hier sagen Sie, wann Sie fahren können.

1. Tippen Sie unten auf **Verfügbar**.
2. Sie sehen die Wochentage: Montag bis Freitag.
3. Zu jedem Tag gibt es Zeit-Fenster von 2 Stunden.
4. Tippen Sie die Fenster an, in denen Sie Zeit haben.
5. Ein angetipptes Fenster wird farbig.
6. Ihre Auswahl wird sofort gespeichert.

> **Wichtig:** Ohne Eintrag gelten Sie an dem Tag als nicht verfügbar.

**Einzelner Tag anders?** Zum Beispiel ein Arzttermin?

1. Gehen Sie zu **Ausnahmen an einzelnen Tagen**.
2. Wählen Sie das Datum.
3. Tragen Sie für diesen Tag die passenden Fenster ein.

Die Ausnahme gilt nur für diesen einen Tag. Sie ersetzt an diesem Tag das normale Wochenraster.

## 3. Ferien oder Abwesenheit beantragen

Hier melden Sie Ferien, Krankheit oder andere freie Tage.

1. Tippen Sie unten auf **Abwesend**.
2. Oben sehen Sie das Formular **Neue Abwesenheit**.
3. Wählen Sie bei **Typ** die Art: Ferien, Krank, Weiterbildung oder Sonstiges.
4. Wählen Sie bei **Von** den ersten Tag.
5. Wählen Sie bei **Bis** den letzten Tag.
6. Schreiben Sie bei Bedarf einen kurzen Grund. Das ist freiwillig.
7. Tippen Sie auf **Abwesenheit beantragen**.

Ihr Antrag geht an die Disposition. Die Disposition prüft ihn.

**Den Stand sehen Sie in der Liste:**

- **Beantragt** – wird noch geprüft.
- **Genehmigt** – ist bewilligt.
- **Abgelehnt** – wurde nicht bewilligt.

**Antrag zurückziehen?** Das geht nur, solange er **Beantragt** ist.

1. Suchen Sie den Antrag in der Liste **Ihre Anträge**.
2. Tippen Sie beim Antrag auf **Stornieren**.

Ist der Antrag schon **Genehmigt**? Dann rufen Sie bitte die Disposition an.

## 4. Kontaktdaten ändern

Hier ändern Sie Ihre Telefonnummer, E-Mail und Adresse.

1. Tippen Sie unten auf **Profil**.
2. Ändern Sie das, was neu ist. Zum Beispiel die Telefonnummer.
3. Tippen Sie auf **Speichern**.

> **Gut zu wissen:** Ihren Namen, das Fahrzeug und den Notfallkontakt kann nur das Büro ändern. Melden Sie sich dafür bitte bei der Disposition.
